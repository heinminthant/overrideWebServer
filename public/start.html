Dialogflow Setup Guide

Step 1 - Creating Dialogflow Account

(You can skip this step if you already have a Dialogflow Account. Go to Step 2 - Create Key File)

Open this link to follow along

You will be greeted with the following page

Press on the Sign Up Button to get to the Sign Up Page

Press Sign In With Google and sign in with your google account

After signing in, you will be redirected to the Welcome Page shown below:

Press on the blue Create Agent Button

Give you agent a name and Press Create

Wait a while for the creation process to finish.

You will then be redirected to the Intents Page shown below:

Congratulations!! You have successfully created a Dialogflow Agent

Step 2 - Getting Key File

Press on the icon on the top left corner to open a sidebar

Then Press on the top right corner of the siderbar

You will arrive at the settings page as provided below:

Press on the service account link

From the list of service accounts, Press the link with the name "Dialogflow Integrations"

Press on the icon on the top right and Press on Edit

Then press the Create Key at the bottom

Choose JSON and press on Create

The key file will be downloaded to your computer

Remember the file name as it will be uploaded in the next step

Step 3 - Upload Key File to Bot

Go to the Telegram Bot

Press on the Dialogflow Menu

Press on the Upload Key File Button

Click on the clip icon on the bottom right

Then click on the File icon

Press on the key file that was downloaded in Step 2

The file will then be uploaded

Great Job. The setup process is finally complete.
